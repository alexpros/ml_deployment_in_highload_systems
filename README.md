# Развёртывание ML-моделей в высоконагруженных системах

## О курсе

В условиях стремительного роста популярности машинного обучения в бизнесе компании сталкиваются с новыми вызовами, связанными с эффективным развёртыванием и управлением ML-моделями.
Основная цель курса — погрузить студентов в современные подходы к развёртыванию ML-решений.
